Who I am and how it all started.

"I believe in proactive security – understanding threats before they become incidents."

I've developed practical skills in log analysis, system security, network monitoring, and investigative troubleshooting. Building security automation and detection tools for file integrity monitoring and log forensics has become my specialty.
